Introducere în C#

C# este un limbaj de programare modern, dezvoltat de Microsoft, folosit în principal pentru aplicații desktop, web și mobile.

De ce să înveți C#?

  • Este ușor de învățat și bine structurat.
  • Are suport puternic pentru OOP (programare orientată pe obiect).
  • Funcționează excelent cu .NET Framework și .NET Core.

Primul tău program C#

using System;

class Program {
    static void Main() {
        Console.WriteLine("Salut lume!");
    }
}

Rularea acestui program va afișa: Salut lume!
